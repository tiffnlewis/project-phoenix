# The Phoenix Arcana Design System

## Design Direction

The Phoenix Arcana blends illuminated manuscripts, tarot cards, antique anatomical illustrations, celestial charts, botanical engravings, and baroque ornamentation.

The interface should feel like a personal wellness grimoire rather than a conventional fitness application.

## Visual Principles

- Light parchment backgrounds
- Dark, readable ink
- Antique-gold borders and ornamentation
- Deep burgundy and forest-green accents
- Strong decorative frames
- Engraved illustrations
- Generous spacing
- Clear mobile hierarchy

## Color Palette

| Token | Value | Purpose |
|---|---:|---|
| Parchment | `#F5EFE2` | Main background |
| Aged Ivory | `#EDE2CC` | Secondary surface |
| Ink | `#2B241C` | Primary text |
| Antique Gold | `#B58B2A` | Borders and emphasis |
| Burgundy | `#7B2D3A` | Warnings and strong accents |
| Forest | `#566648` | Success and recovery |
| Blue Steel | `#495C73` | Informational accent |

## Typography

### Display

Cinzel Decorative

Used for:

- App title
- Tarot card titles
- Major section headings

### Heading

Cormorant Garamond

Used for:

- Page headings
- Exercise names
- Subheadings

### Body

Inter

Used for:

- Instructions
- Forms
- Buttons
- Accessibility text

## Layout

- Mobile-first
- Maximum content width of approximately 480px
- Persistent bottom navigation
- Large tap targets
- Clear vertical reading flow
- Standing exercises before floor exercises

## Borders and Ornamentation

Use ornamental borders to create hierarchy.

Frame types:

- Page frame
- Hero frame
- Tarot card frame
- Notice frame
- Button frame
- Modal frame

Borders should feel detailed but must not reduce readability.

## Buttons

Primary buttons:

- Forest or dark ink background
- Antique-gold border
- Light parchment text
- Clear pressed and focus states

Secondary buttons:

- Parchment background
- Antique-gold border
- Ink text

## Cards

Tarot-style cards should include:

- Tall proportions
- Decorative corner marks
- Centered title
- Engraved illustration
- Exercise details
- Clear completion state

## Timer

The timer uses an hourglass metaphor.

Required states:

- Ready
- Running
- Paused
- Complete
- Reset

The countdown must still display numerically for accessibility.

## Navigation

Primary destinations:

- Oracle
- Quest
- Grimoire
- Ledger
- Archive

Each destination should include both an icon and a text label.

## Motion

Animations should be subtle:

- Page fades
- Gentle card movement
- Hourglass sand movement
- Border shimmer used sparingly
- Optional page-turn transition

Avoid distracting movement during exercise.

## Accessibility

- Maintain readable contrast
- Do not rely on color alone
- Use visible focus states
- Support keyboard navigation
- Keep body text comfortably sized
- Include reduced-motion support
- Pair decorative labels with plain-language instructions

## Tone

The visual atmosphere should evoke:

- Ancient scholarship
- Personal ritual
- Natural history collections
- Celestial observation
- Quiet encouragement

Avoid:

- Halloween imagery
- Horror styling
- Excessively dark backgrounds
- Hard-to-read decorative fonts
- Overcrowded screens
